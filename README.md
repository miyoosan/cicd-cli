# vcicd-cli

```shell
# install
yarn global add vcicd-cli

cicd --help
----
Usage:  [options] [command]

Options:
  -V, --version  output the version number
  -h, --help     output usage information

Commands:
  jenkins        创建Jenkins多分支流水线任务
  gitlab         添加webhooks到gitlab仓库
  script         为项目生成CI脚本
  init           自动配置CI流程
```
